# Aip Raidió Fáilte

<p>Aip ghréasáin fhorásach é seo a forbríodh do Raidió Fáilte 107.1fm, stáisiún raidió pobail Gaeilge Bhéal Feirste. Táthar in ann éisteacht beo leis an stáisiún, éisteacht siar ar chláracha, dul i dteagmháil, agus eolas a fháil ar an aip. Cruthaíodh dhá fhréamhshamhail don aip ar dtús - PWA, agus leagan le React Native. Deineadh an cinneadh leanúint ar aghaidh leis an leagan seo agus úsáideadh é i gcomhair an táirge chríochnaithe.</p>

<p><i>Progressive web application developed for Belfast Irish language community radio station Raidió Fáilte 107.1fm. It has a live stream, listen back section, contact page, and an information page. Two prototypes were initially made - a PWA and a React Native version. This PWA was developed further and used for the production app.</i></p>
